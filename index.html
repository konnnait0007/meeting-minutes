使い方の流れ

  1. 初回のみ「⚙ 設定」から AIプロバイダーとAPIキーを登録
  2. 同じく設定の「匿名化リスト」に利用者様・職員のお名前を登録
  3. 会議の種類のタブを選択（全体会／支援会議／企画会議）
  4. STEP1: 期日・出席者などの基本情報を入力
  5. STEP2: 会議内容を入力（3つの方法から選択）
    • テキスト貼付: Teams等の文字起こしを貼る（一番手軽）
    • ライブ録音: 会議中にPCで録音→自動文字起こし
    • 音声ファイル: 録音済みMP3等をアップロード（要OpenAIキー）
  6. STEP3: 「会議録を生成」ボタンを押す
  7. STEP4: 編集して Word 形式でダウンロード

プライバシーについて

このアプリはサーバーを持たず、すべてブラウザ内で動作します。 APIキーや匿名化リストはお使いのPCのlocalStorageに保存され、外部送信されません。
AIに送る前に「匿名化リスト」の名前は伏字に置換され、応答が返ってきた時点で元の名前に復元します。 ただし、固有名詞のすべてを匿名化できるわけではないため、出力結果は必ず人の目で確認してください。

無料で使うには

要約 → Gemini API: 日次の無料枠あり（モデルにより制限）。
音声ファイル文字起こし → Groq API: メアド登録のみで無料・高速（Whisper-large-v3）。 console.groq.com/keys で取得。
会議中のライブ録音 → Web Speech API: ブラウザ標準機能でAPIキー不要。
120分のスマホ録音もブラウザ内で自動的に16kHzモノラルへ圧縮・分割するので、25MB制限を意識する必要はありません。

トラブル

  • 「APIキーが無効」→ 設定でキーを確認、コピー時の余分な空白に注意
  • 録音が止まる → ブラウザのマイク許可と、PCの省電力設定を確認
  • 長すぎる会議 → STEP2のテキストを2分割して2回生成し、後でつなげる
